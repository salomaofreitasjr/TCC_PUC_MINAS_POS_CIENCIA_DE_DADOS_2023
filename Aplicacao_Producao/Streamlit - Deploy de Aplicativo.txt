    - listar versões de bilbiotecas utilizadas no projeto, exceto streamlit
	- verificar no terminal no VsCode as versões das nossas bibliotecas que estão no import, com o comando:
		pip freeze show  ou pip pandas show


2- Colocar nosso projeto em um repositório no github (OK, já temos)

3- Logar no streamlit (o meu está para logar com a conta do github)

		- Preencher as informações 
		- Colocar uma URL boa para a aplicação 
		- Em Advanced Settings, colocar a versão do Python que usamos (no nosso caso foi a 3.10)
		- Clicar Botão Deploy
